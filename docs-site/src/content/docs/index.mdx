## The one thing that's different

Every other agent framework ships agents you debug with **logs**. `agentfootprint` ships agents you debug with a **time-travel slider**.

```ts
import { Agent, exportTrace, anthropic } from 'agentfootprint';

const agent = Agent.create({ provider: anthropic('claude-sonnet-4') })
  .system('You are a helpful assistant.')
  .tools([...])
  .build();

await agent.run('What is 4 + 5 + 67?');

// Capture the full execution as portable JSON
const trace = exportTrace(agent);
console.log(JSON.stringify(trace));
```

Paste that JSON into the [viewer](https://footprintjs.github.io/agent-playground/#/viewer) → see the flowchart, the narrative timeline, the memory state at every step. No extra SaaS, no hosted service, no instrumentation. The trace is already there because every stage records itself as it runs.

## 30-second demo

1. Pick a pattern in [Live Chat](https://footprintjs.github.io/agent-playground/#/live) (works with a mock adapter — no API key needed).
2. Send a message. See the response + the **Behind the Scenes** button.
3. Click **Copy Trace**. Paste it into [Trace Viewer](https://footprintjs.github.io/agent-playground/#/viewer).
4. Drag the time-travel slider. Every step of the agent's reasoning is there.

Zero typing. Zero keys. Everything else in this library follows from this one workflow.

## Install

```sh
npm install agentfootprint footprintjs
```

Peer dep: `footprintjs >= 4.14.0`. Optional: `@anthropic-ai/sdk`, `openai`, `@aws-sdk/client-bedrock-runtime` — depending on the LLM provider you want.
